# Тестовое задание "Калькулятор"

### Приложение разделено на 4 части:

- **App** (входная точка и реализация сохранений) - имеет доступ ко всем другим частям приложения
- **Core** (содержит правила валидации и вычисления выражений, модель данных выражения, константы, интерфейсы сохранений) - не имеет доступа к другим частям приложения
- **Presenter** (реализует взаимодействие между Core и UI, содержит интерфейсы для view кальтулятора и окна информационных сообщений) - имеет доступ только к Core
- **UI** (разделено на 2 части - окно калькулятора и окно информационных сообщений, содержит все отображаемые элементы приложения, а также некоторые вспомогательные, которые относятся только к отображению) - имеет доступ только к Presenter

### Комментарии:
- в приложении не предусмотрен ввод длинных чисел, не помещающихся в диапазон int - при вводе таких чисел будет ошибка Unity
- в приложении предусмотрено множественное сложение, например, 23+341+49
- в меню редактора добавлена опция *Tools -> Clear PlayerPrefs* для удаления сохранений
- cцена для запуска: Assets -> _Project -> Scenes -> Main.unity


Сделано в версии Unity 2022.3.36f1
